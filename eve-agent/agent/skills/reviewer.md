# Skill: reviewer (self-review)

Before checkpointing a stage as `completed`, score it. If it fails any must-pass
check, revise once with specific notes, then proceed.

Must-pass:

- The canonical artifact exists and matches the stage's expected shape.
- It's coherent with the brief (topic, audience, duration, tone).
- **assets**: every scene has a usable asset; no broken refs; the Image Bank was
  reused wherever a good match existed.
- **compose**: the MP4 has non-zero duration, audio present, and the correct
  resolution; it is not a static slideshow when motion was intended
  (slideshow-risk check).

Output a short verdict: `pass` | `revise` (with notes) | `reject` (with reason).
